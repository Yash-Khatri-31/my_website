Face-Detector

An interactive react app which detects the face present in an image. Begins with a signin/register page for the user. Each email is treated as a unique email to prevent repetition. Followed by a page where the user enters their jpg url of the face which then updates the score of the user. Created various components for ease of understanding. Used clarifai face detection model. For the backend nodejs and expressjs were used. The api was connected to the database using knexjs. All the 3 different areas (frontend, backend and database) were uploaded on git and heroku was used to make the site go live.

Nasa Project

Created a React app where the user enters details to initiate a launch to a planet in the coming future. We used Kepler’s data from NASA Exoplanet Archive. Using express we parsed the csv file and created a readable stream where we processed on the data to filter out the exoplanets based on certain criteria. Applied the MVC framework, created separate folders for the model, controller, and routes. Created APIs’ which were responsible for getting the launches from the frontend and storing them in the server side variable. Other API was dedicated for deleting the launches if they were cancelled by the user and adding them into the history page. With the help of postman, we first confirmed the working of each API by hard coding details. Later, used the details from frontend to make the require changes. Used GET, POST, and DELETE methods for this project.
